Steps to run the two-phase commit demo with 3 processes:

1. Install python v3.9
2. Navigate to directory where Coordinator, Process_A, and Process_B files are present
3. Run command: python3 Process_A.py
4. Run command: python3 Process_B.py
5. Run command: python3 Coordinator.py
6. Insert details and see output on each process window
